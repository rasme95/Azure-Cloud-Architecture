## Overview

This repository contains a cloud architecture proposal developed as part of the Cloud Computing programme at **Noroff University College**.

The project presents an enterprise hybrid cloud architecture for the fictional company **NerdyClouds**, designed around Microsoft Azure and modern cloud architecture principles.

The accompanying PDF documents the complete architecture, design decisions and Proof of Concept validation.
---

## Architecture Scope

| Domain | Technologies |
|---------|--------------|
| Cloud Platform | Microsoft Azure |
| Networking | Hub-and-Spoke, VNet Peering, VPN Gateway, Private DNS |
| Identity | Microsoft Entra ID, Hybrid Identity, RBAC, Conditional Access |
| Security | Zero Trust, Least Privilege, MFA |
| Platform Services | App Service, Blob Storage, Azure Monitor |
| Data & Analytics | Microsoft Fabric, Power BI |
| AI | Azure AI Language, Microsoft Foundry |
